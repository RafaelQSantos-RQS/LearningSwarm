# docker swarm

Gerencia o Swarm.

## Uso

```bash
docker swarm [COMMAND]
```

## Subcommands

| Comando | Descrição | Arquivo |
|---------|-----------|---------|
| `docker swarm init` | Inicializa um swarm | [init](./02_docker_swarm_init.md) |
| `docker swarm join` | Junta um nó ao swarm | [join](./03_docker_swarm_join.md) |
| `docker swarm join-token` | Gerencia tokens de entrada | [join-token](./04_docker_swarm_join-token.md) |
| `docker swarm leave` | Sai do swarm | [leave](./05_docker_swarm_leave.md) |
| `docker swarm ca` | Exibe e rotaciona a CA raiz | [ca](./01_docker_swarm_ca.md) |
| `docker swarm unlock` | Desbloqueia o swarm | [unlock](./06_docker_swarm_unlock.md) |
| `docker swarm unlock-key` | Gerencia chave de desbloqueio | [unlock-key](./07_docker_swarm_unlock-key.md) |
| `docker swarm update` | Atualiza o swarm | [update](./08_docker_swarm_update.md) |
